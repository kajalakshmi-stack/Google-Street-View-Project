# Google Street View Image Classification using Deep Learning and Computer Vision

## Project Overview

This project presents an advanced Computer Vision solution for automated analysis and classification of Google Street View imagery using Deep Learning techniques. The system leverages Convolutional Neural Networks (CNNs) implemented with TensorFlow and Keras to extract meaningful visual patterns from street-level images and perform accurate image classification.

The objective of this project is to demonstrate the practical application of Artificial Intelligence, Computer Vision, and Deep Learning in real-world image recognition tasks. The model is trained on a curated Street View dataset and deployed through an interactive Streamlit web application, enabling real-time image prediction and analysis.

---

## Key Objectives

* Develop an intelligent image classification system using Deep Learning.
* Apply Computer Vision techniques for feature extraction and pattern recognition.
* Train and evaluate a Convolutional Neural Network (CNN) on Street View imagery.
* Deploy the trained model through a user-friendly web interface.
* Demonstrate the integration of Machine Learning and Web Technologies for practical applications.
